# M4 Hunt: 16/3 Ratio in CP² × S³ GW Spectral Mode Counting

**Source:** `Constitutive_Derivation_of_Tensor_Gravitational_Radiation_from_CP_2___S3_Spectral_Geometry_in_Density_Field_Dynamics.pdf` (Alcock, April 2026, 11 pp.)

## Verdict

**NO 16/3 ratio appears anywhere in this paper.** The spectral mode counting on CP² × S³ does not produce a 16/3 partitioning between any sectors.

## Evidence by sub-question

### (a) Transverse vs longitudinal mode count
The 3+1 O(3) decomposition (Eq. 6) gives:
- spin-0 (trace ψ): **1 DOF**
- spin-2 (hTT): **2 DOF**
- gauge/constrained: vector + scalar (non-propagating)

Ratio of propagating tensor to scalar = **2/1**, not 16/3. The full low-energy spectrum table (§5) lists 1 scalar + 2 tensor + (8+3+1)=12 gauge + 1 Planck-massive squashing modulus. None of these combine into 16/3.

### (b) S³ vs CP² mode multiplicity
- S³ Lichnerowicz TT spectrum (Thm. 4.2): λ_ℓ = ℓ(ℓ+n−1) − 2 + 2n; for n=3, λ_min = **12**.
- CP² Lichnerowicz TT spectrum (Thm. 4.4 / Besse): primitive (1,1) gap = 4n = **8**; (2,0)+(0,2) gap = 8(n+1) = **24**; CP² gap λ_min = **8**.

Ratios available: 12/8 = 3/2, 24/8 = 3, 12/24 = 1/2. **No 16/3.**

Dimensions: dim CP² = 4, dim S³ = 3, dim K = 7. Ricci eigenvalues: ĝ → 6, ǧ → 2 (ratio 3). Volumes: π²R₁⁴/2 and 2π²R₂³. **No 16/3.**

### (c) 16 tensor DOF / 3 generation partitioning
The paper makes **no reference to 16 tensor DOF nor to 3 fermion generations** in the GW context. The propagating tensor sector has 2 DOF (graviton polarizations), not 16. Generation structure is not invoked anywhere in this derivation. The (3,2,1) Standard Model partition is mentioned only in passing (κ = α/n₂² with n₂ = 2 the SU(2) frame stiffness, §7) and unrelated to a 16/3 split.

### (d) Trace anomaly coefficients on the internal manifold
The Seeley–DeWitt heat-kernel coefficients a₀, a₂, a₄ are referenced (§1, §6) as the source of cosmological constant, gauge kinetic, and Einstein–Hilbert terms — but **no explicit trace anomaly coefficient values are given**, and certainly nothing producing 16/3. Curvature invariants computed (Eq. 4): R_K = 24/R₁² + 6/R₂², |Ric_K|² = 144/R₁⁴ + 12/R₂⁴. Coefficient ratios here are 24:6 = 4:1 and 144:12 = 12:1. **No 16/3.**

## Notable ratios that DO appear
- **1/4** — relative scalar/tensor effective-action coefficient (§6): K₀ = c⁴/(8πG) compression vs K₀/4 shear stiffness. This is the central quantitative ratio of the paper.
- **1/√3** — the squashing modulus fixed point τ* = R₂/R₁ (Thm. 4.9), the Einstein product condition.

## Conclusion
The paper's mode counting on CP² × S³ is fully accounted for by (1 scalar + 2 tensor) propagating + 12 gauge + 1 Planck-massive modulus, with the only structural ratio being the 1:4 compression-to-shear stiffness from Einstein–Hilbert linearization. **There is no 16/3 ratio between transverse/longitudinal sectors, between S³ and CP² mode multiplicities, between tensor DOF and generations, or in any trace-anomaly coefficient given in this document.**
