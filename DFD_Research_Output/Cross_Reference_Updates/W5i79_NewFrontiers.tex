\section{New Research Frontiers from i79}

\subsection{Frontier 1: Post-Publication Community Engagement}
Once papers are public, the programme transitions from development to engagement:
\begin{itemize}[nosep]
\item \textbf{arXiv response papers}: Expect 5--15 response/comment papers within 6 months. Need rapid-response capability.
\item \textbf{Seminar circuit}: Target 10--15 university seminars in Year 1.
\item \textbf{Online presence}: Dedicated DFD website with prediction tracker, Zenodo links, FAQ.
\item \textbf{Blog/explainer}: Non-technical summary for ScienceNews, Quanta Magazine, etc.
\end{itemize}

\subsection{Frontier 2: DFD Software Ecosystem}
Beyond JOSS paper, a broader software ecosystem is needed:
\begin{itemize}[nosep]
\item \textbf{DFD-CLASS}: Modified CLASS (Boltzmann solver) with DFD $\psi$-coupling. For CMB and LSS forecasts.
\item \textbf{DFD-CAMB}: Alternative Boltzmann solver implementation.
\item \textbf{DFD-N-body}: Public fork of AREPO or GADGET-4 with DFD force law.
\item \textbf{DFD-Fisher}: Fisher matrix code for multi-probe forecasting.
\item \textbf{DFD-MCMC}: CosmoMC/MontePython plugin for DFD parameter estimation.
\end{itemize}

\subsection{Frontier 3: Second-Generation Predictions}
With v3.3 submitted, the next theoretical frontier is second-generation predictions:
\begin{itemize}[nosep]
\item \textbf{DFD at 2-loop}: Higher-order corrections to $\alpha$, fermion masses, BAO.
\item \textbf{DFD + neutrino physics}: Full 3-flavour oscillation from $\CP^2$ geometry.
\item \textbf{DFD + dark matter}: If DFD is correct, what is dark matter? (DFD does not eliminate DM; it modifies gravity but still requires $\Omega_{\rm DM} \approx 0.26$.)
\item \textbf{DFD strong sector}: QCD $\beta$-function from $\CP^2$ spectral action.
\end{itemize}

\subsection{Frontier 4: International Collaboration Network}
The 8 identified collaborators from i79 represent the seed of a DFD research network:
\begin{itemize}[nosep]
\item \textbf{Theory node}: Spectral geometry, FRG, noncommutative geometry.
\item \textbf{Simulation node}: N-body, hydrodynamics, emulators.
\item \textbf{Data node}: Euclid, DESI, LIGO, CMB-S4 analysis.
\item \textbf{Mathematics node}: Toeplitz operators, spectral triples, algebraic topology.
\end{itemize}
