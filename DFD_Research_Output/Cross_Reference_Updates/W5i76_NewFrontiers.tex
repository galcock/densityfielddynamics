\subsection{Frontier 1: N-Body Halo Statistics}
The first DFD N-body results reveal a $+1.5\%$ excess in the halo mass function at $M \sim 10^{14}\,M_\odot$. This opens:
\begin{itemize}[nosep]
\item Void-halo cross-correlation in DFD (voids expand slightly faster due to $\psi$-enhancement)
\item Cluster concentration-mass relation ($c$--$M$: DFD predicts $\sim 2\%$ lower concentrations at $M > 10^{14}\,M_\odot$ due to enhanced tidal stripping)
\item Splashback radius: DFD predicts $R_{\rm sp}$ shifted outward by $\sim 0.5\%$ (testable with DES Y6)
\end{itemize}

\subsection{Frontier 2: $E_G$ Cross-Correlation Optimisation}
The $E_G$ Fisher forecast reveals that the DFD signal is maximised in the cross-correlation of:
\begin{itemize}[nosep]
\item Euclid photometric lensing $\times$ DESI BGS ($z = 0.2$--$0.4$): highest $S/N$ per bin
\item 4MOST + Euclid ($z = 0.5$--$0.8$): complementary spectroscopic coverage
\item CMB lensing $\times$ spectroscopic: independent systematic profile
\end{itemize}

\subsection{Frontier 3: Multi-Messenger GW--EM Synergy}
DFD's unique prediction ($D_L^{\rm GW}/D_L^{\rm EM} = 1$ exactly) enables a clean null test. With O5:
\begin{itemize}[nosep]
\item 50--100 BNS events with EM counterpart expected (O5 + Rubin)
\item Each event constrains $\delta(D_L^{\rm GW}/D_L^{\rm EM}) \sim 5\%$
\item Stacked analysis: $\sigma \sim 0.5\%$ on $D_L$ ratio, sufficient to distinguish DFD from $f(R)$
\end{itemize}

\subsection{Frontier 4: CMB-S4 Inflation Predictions}
DFD slow-roll predictions form a tight cluster in the $n_s$--$r$ plane:
\begin{itemize}[nosep]
\item $n_s = 0.9649 \pm 0.0003$ (DFD), vs $0.9649 \pm 0.0042$ (Planck 2018)
\item $r = 0.0040 \pm 0.0008$ (DFD), vs $< 0.036$ (BICEP/Keck 2021)
\item Running: $\alpha_s = -5.3 \times 10^{-4}$ (DFD), vs $-0.006 \pm 0.013$ (Planck)
\item CMB-S4 will measure $r$ to $\sigma(r) = 0.001$: $4\sigma$ detection if DFD correct
\end{itemize}
