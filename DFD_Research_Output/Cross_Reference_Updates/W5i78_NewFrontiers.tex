\subsection{Frontier 1: Post-Adversarial Strengthening}
The comprehensive audit identified areas where DFD's case could be strengthened further:
\begin{itemize}[nosep]
\item \textbf{$m_s$ lattice comparison}: Write a dedicated comparison paper matching DFD prediction against all major lattice QCD collaborations (MILC, BMW, RBC/UKQCD, HPQCD). Show that DFD is within the inter-collaboration spread.
\item \textbf{$\alpha$ convergence analysis}: Publish the spectral action convergence sequence ($n = 1, 2, 3, 4$) explicitly. Show that the series converges and that $n = 5$ correction is bounded.
\item \textbf{$k = 60$ uniqueness}: Develop the homotopy argument for $k = |\pi_7(S^4)| = 60$ into a standalone mathematical paper.
\end{itemize}

\subsection{Frontier 2: Einstein Telescope Era}
The multi-messenger paper's ET sensitivity curves reveal extraordinary potential:
\begin{itemize}[nosep]
\item ET will detect $\sim 10^5$ BNS mergers per year. With $\sim 1\%$ having EM counterparts: $\sim 1000$ events.
\item Stacked $D_L^{\rm GW}/D_L^{\rm EM}$: $\sigma = 0.04\%$. This is 100$\times$ better than O5.
\item \textbf{Definitive DFD test}: if $D_L^{\rm GW} \neq D_L^{\rm EM}$ at $> 5\sigma$, DFD is falsified. If equal, DFD + GR are consistent.
\item \textbf{GW lensing search}: ET rate of multiply-lensed GW events $\sim 10$/yr. Definitive test of DFD GW no-lensing theorem within first year of operation.
\end{itemize}

\subsection{Frontier 3: DESI Y3 Analysis Pipeline}
With pre-registered predictions finalised, the next step is to build an automated analysis pipeline:
\begin{itemize}[nosep]
\item Script to download DESI Y3 data products on release day.
\item Automatic $\chi^2$ computation against DFD predictions.
\item Traffic-light classification: GREEN ($< 2\sigma$), YELLOW ($2$--$3\sigma$), RED ($> 3\sigma$).
\item Public notebook for reproducibility.
\end{itemize}

\subsection{Frontier 4: N-Body Phase 2 Planning}
Current N-body (Phase 1: $L = 500$, $N = 1024^3$) nearing completion. Phase 2 requirements:
\begin{itemize}[nosep]
\item $L = 1000\,h^{-1}$Mpc for BAO-scale predictions ($\sim 800$k CPU-hours)
\item Cosmology variation suite: 5 values of $\Omega_m$ ($\sim 400$k CPU-hours per)
\item Baryonic feedback implementation ($\sim 200$k CPU-hours)
\item Total Phase 2: $\sim 3$M CPU-hours. Requires major HPC proposal.
\end{itemize}
