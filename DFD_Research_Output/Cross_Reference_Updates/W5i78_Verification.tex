\section{Adversarial Audit Methodology}

Each of the 12 auditing agents (Agents 1--12) applied the following protocol to their assigned GREEN items:

\begin{enumerate}[nosep]
\item \textbf{Re-derivation}: Attempt to derive the result from first principles, without referring to existing derivation files.
\item \textbf{Parameter sensitivity}: Vary key inputs by $\pm 10\%$ and check if the result changes qualitatively.
\item \textbf{Logical gap search}: Identify every assumption and check if each is (a) stated, (b) justified, (c) necessary.
\item \textbf{Cross-check}: Compare result against at least one independent source (external literature, different calculation method, or numerical verification).
\item \textbf{Adversarial scenario}: Construct the strongest possible argument against the result and attempt to refute it.
\end{enumerate}
