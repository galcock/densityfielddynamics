## 1. Standard Perturbation Theory (SPT): The F2 and G2 Kernels

### 1.1 The Framework

The canonical reference is **Bernardeau, Colombi, Gaztanaga & Scoccimarro (2002)**, "Large-Scale Structure of the Universe and Cosmological Perturbation Theory," Physics Reports 367, 1-248 [arXiv:astro-ph/0112551]. This 248-page review is the bible of cosmological perturbation theory.

In SPT, the density contrast delta and velocity divergence theta are expanded perturbatively:

```
delta = delta^(1) + delta^(2) + delta^(3) + ...
theta = theta^(1) + theta^(2) + theta^(3) + ...
```

At each order n, the solution in Fourier space is:

```
delta^(n)(k) = integral [d^3 k_1 ... d^3 k_n] delta_D(k - k_1 - ... - k_n)
               x F_n(k_1, ..., k_n) delta_0(k_1) ... delta_0(k_n)
```

where F_n are the SPT kernels (similarly G_n for theta).

### 1.2 The F2 Kernel (Critical for Mode Coupling)

The second-order kernel takes the explicit form:

```
F_2(k_1, k_2) = 5/7 + (1/2)(k_1.k_2)(1/k_1^2 + 1/k_2^2) + (2/7)(k_1.k_2)^2/(k_1^2 k_2^2)
```

This kernel encodes HOW two linear modes at k_1 and k_2 combine to generate power at k = k_1 + k_2. The three terms represent:
- **5/7**: isotropic growth (independent of geometry)
- **Tidal term**: depends on alignment of the two wavevectors
- **2/7 term**: quadrupolar coupling

The G2 kernel for the velocity divergence is:

```
G_2(k_1, k_2) = 3/7 + (1/2)(k_1.k_2)(1/k_1^2 + 1/k_2^2) + (4/7)(k_1.k_2)^2/(k_1^2 k_2^2)
```

### 1.3 Power Generated by Mode Coupling

The one-loop power spectrum has two contributions:

```
P_1-loop(k) = P_22(k) + 2 P_13(k)
```

where:

```
P_22(k) = 2 integral d^3q |F_2(q, k-q)|^2 P_L(q) P_L(|k-q|)
```

This is a convolution integral -- it takes power from ALL pairs of modes that add to k. The integrand is sharply peaked around q ~ k/2, meaning mode coupling primarily transfers power from modes at roughly half the scale.

```
P_13(k) = 3 P_L(k) integral d^3q F_3(k, q, -q) P_L(q)
```

This term modifies the existing power at k by coupling with all modes q.

**KEY POINT FOR DFD**: In standard gravity, all of this is O(delta^2) -- mode coupling is a SECOND-ORDER effect. The first-order solution is simply linear growth with no coupling. This is because the Poisson equation nabla^2 Phi = 4piG rho is LINEAR.

### 1.4 References for SPT

- Crocce & Scoccimarro (2006) on Renormalized Perturbation Theory (RPT)
- McQuinn & White (2016) on FAST-PT algorithm for computing loop integrals [arXiv:1609.05978]

## 2. Mode Coupling in MOND / Modified Gravity

### 2.1 The AQUAL Field Equation

The AQUAL (AQUAdratic Lagrangian) formulation by Bekenstein & Milgrom (1984) gives the field equation:

```
nabla . [mu(|nabla Phi| / a_0) nabla Phi] = 4 pi G rho
```

This is structurally identical to the DFD field equation:

```
nabla . [mu(|nabla psi| / a*) nabla psi] = -(8piG/c^2) rho
```

The crucial difference from Newtonian gravity: **the operator itself is nonlinear**. In Newtonian gravity, nabla^2 is a linear operator acting on Phi, and nonlinearity enters only through the source (rho depends on delta which depends on Phi through dynamics). In MOND/DFD, the differential operator contains mu which depends on nabla Phi itself.

### 2.2 Llinares, Knebe & Zhao (2008)

**"Cosmological structure formation under MOND: a new numerical solver for Poisson's equation"**, MNRAS 391, 1778 [arXiv:0809.2899]

Key findings:
- Standard FFT-based N-body codes CANNOT solve the MOND Poisson equation because it is nonlinear
- They developed a multigrid relaxation solver within the AMIGA N-body code
- The nonlinearity means the superposition principle fails: the total potential is NOT the sum of individual source potentials
- This is computational evidence that mode coupling in MOND is fundamentally different from Newtonian gravity

### 2.3 Nusser (2002)

**"Modified Newtonian dynamics of large-scale structure"**, MNRAS 331, 909

Key findings:
- Used N-body simulations with a heuristic MOND prescription
- If the Newtonian initial density field has a power-law spectrum P(k) ~ k^n with n < -1, then MOND dominance proceeds from small to large scales
- **MOND tends to drive the density power spectrum towards k^{-1}** -- this is a spectacular result suggesting the nonlinearity has a universal attractor
- Standard MOND acceleration a_0 = 10^{-8} cm/s^2 gives too much clustering; a_0 ~ 10^{-9} cm/s^2 matches observations better

**CRITICAL IMPLICATION FOR DFD**: The tendency toward P(k) ~ k^{-1} suggests the MOND nonlinearity redistributes power across scales in a specific, predictable way. This could be the signature of operator-level mode coupling.

### 2.4 nuHDM Simulations (2025)

**"How does a MOND cosmology fare on Gpc scales?"**, MNRAS 547 [arXiv:2602.21975]

The largest collisionless N-body simulations in MOND cosmology to date, using the nuHDM (neutrino hot dark matter) framework:
- The model can reproduce the CMB power spectrum
- **MOND models massively overproduce large-scale structures**
- This overproduction is likely connected to the enhanced mode coupling from the nonlinear operator

### 2.5 QUMOND: The Quasi-Linear Shortcut

**Milgrom (2010)**, "Quasi-linear formulation of MOND", MNRAS 403, 886

QUMOND reformulates MOND so that only the standard linear Poisson equation needs to be solved (twice), with nonlinear steps being purely algebraic. This is computationally much cheaper but conceptually important:

- QUMOND separates the "linear propagation" from the "nonlinear modification"
- The phantom dark matter distribution can be computed from the Newtonian potential
- For DFD perturbation theory, QUMOND suggests a perturbative strategy: solve linearly first, then apply nonlinear corrections algebraically

### 2.6 Simulation Techniques for Modified Gravity

**Llinares (2021)**, "Simulation techniques for modified gravity" [arXiv:2103.10890]

Reviews numerical methods for modified gravity cosmological simulations, emphasizing that nonlinear field equations generally require multigrid solvers and that the jump in complexity from linear to nonlinear PDEs is substantial.

## 3. The Key Difference: O(delta) vs O(delta^2) Mode Coupling

### 3.1 The Argument

In standard Newtonian gravity:
- The Poisson equation nabla^2 Phi = 4piG rho_bar (1 + delta) is **linear in Phi**
- Mode coupling arises from the **continuity and Euler equations** which are nonlinear in (delta, v)
- Therefore mode coupling is O(delta^2): you need TWO perturbations to interact

In MOND/DFD with mu(x) = x/(1+x):
- The field equation nabla . [mu(|nabla psi|/a*) nabla psi] = source is **nonlinear in psi already at the operator level**
- Even for a SINGLE Fourier mode delta(k), the solution psi(k) involves contributions from other k-modes
- Therefore mode coupling could be O(delta): even a FIRST-ORDER perturbation couples modes

### 3.2 Explicit Demonstration

Consider linearizing the MOND operator around a background field nabla psi_0:

```
mu(|nabla psi|/a*) nabla psi  ≈  mu_0 nabla delta_psi + mu'_0 (nabla psi_0 . nabla delta_psi / |nabla psi_0|) nabla psi_0 / a*
```

where delta_psi = psi - psi_0, mu_0 = mu(|nabla psi_0|/a*), mu'_0 = dmu/dx evaluated at x = |nabla psi_0|/a*.

The SECOND TERM is the mode coupling term. It depends on the DIRECTION of nabla psi_0 relative to nabla delta_psi, meaning:
- The linearized operator is **anisotropic** -- it depends on the background field direction
- Different Fourier modes of delta_psi see different effective operators depending on their alignment with the background
- **This is first-order mode coupling**: a single perturbation mode interacts with the background to produce anisotropic response

### 3.3 Has Anyone Shown This Explicitly?

**Partial results exist but no complete treatment has been published:**

- The anisotropy of the linearized MOND operator is well known in galaxy dynamics (the "external field effect")
- Bose gas collective excitations in MOND show anisotropic dispersion relations arising from the nonlinear Poisson equation [arXiv:2601.12848]
- **No paper has explicitly computed the mode-coupling kernel analogous to F_2 for a MOND/AQUAL Poisson equation in a cosmological context**

**THIS IS A GAP IN THE LITERATURE THAT DFD COULD FILL.**

## 4. Nonlinear Transfer Functions and BAO

### 4.1 Standard Picture

In LCDM, the nonlinear power spectrum differs from the linear one primarily at small scales (k > 0.1 h/Mpc). The key effects:

- **Mode coupling transfers power to small scales**: P_22 adds power at high k
- **BAO features are damped**: mode coupling generates oscillations out of phase with linear BAO, partially canceling them. Crocce & Scoccimarro (2008) showed this in detail [arXiv:0704.2783]
- **Acoustic peak shifts**: in the correlation function, mode coupling induces percent-level shifts in the BAO peak position
- Renormalized Perturbation Theory (RPT) and Regularized PT handle this by resumming the perturbative series

### 4.2 MOND Nonlinearity and BAO

The MOND nonlinearity could affect BAO in qualitatively different ways:

1. **Enhanced damping**: If mode coupling is O(delta) rather than O(delta^2), BAO features could be damped more efficiently
2. **Scale-dependent transfer**: The mu function introduces a characteristic scale (a*), so the "transfer" of power is not scale-free
3. **Power redistribution**: Nusser's finding that MOND drives P(k) toward k^{-1} suggests systematic redistribution of power from BAO scales to other scales

**No published calculation exists for how a MOND-like nonlinear Poisson equation specifically affects BAO features.** This is another literature gap.

### 4.3 Skordis & Zlosnik (2021) -- The Closest Approach

**"New Relativistic Theory for Modified Newtonian Dynamics"**, Phys. Rev. Lett. 127, 161302 [arXiv:2007.00082]

This relativistic MOND theory (called "AeST") achieves:
- CMB temperature power spectrum indistinguishable from LCDM
- Compatible matter power spectrum at linear level
- But the nonlinear power spectrum and BAO evolution have not been computed in this framework

## 5. Mathematical Structure of the Nonlinear Solution Operator

### 5.1 Why It Is Not a Convolution

For the linear Poisson equation nabla^2 Phi = 4piG rho:
- In Fourier space: -k^2 Phi(k) = 4piG rho(k)
- Solution: Phi(k) = -4piG rho(k) / k^2
- The Green's function is G(k) = -4piG/k^2 and the solution is a convolution in real space

For the nonlinear equation nabla . [mu(|nabla Phi|) nabla Phi] = S:
- In Fourier space, mu depends on nabla Phi which is itself a function of ALL Fourier modes
- The operator is NOT diagonal in Fourier space
- The solution cannot be written as Phi(k) = G(k) S(k) for any function G

### 5.2 Mathematical Structure

The solution operator for the nonlinear Poisson equation has the structure of a **nonlinear integral operator**:

```
psi(k) = K_1(k) rho(k) + integral K_2(k; q) rho(q) rho(k-q) d^3q
         + integral integral K_3(k; q_1, q_2) rho(q_1) rho(q_2) rho(k-q_1-q_2) d^3q_1 d^3q_2 + ...
```

where K_1 is the linear Green's function (modified by mu evaluated on the background) and K_2, K_3, ... are mode-coupling kernels analogous to SPT's F_2, F_3, ...

**BUT CRUCIALLY**: Unlike SPT where the F_n kernels are universal (they don't depend on the power spectrum), the K_n kernels for the nonlinear Poisson equation depend on the **background field configuration**. This means:
- The kernels are environment-dependent
- The perturbative expansion is around a specific background, not around zero
- The expansion may not converge well in the deep-MOND regime

### 5.3 Pseudo-Spectral Methods

For numerical computation, pseudo-spectral methods are the standard approach:
- Compute nabla psi in real space via FFT
- Compute mu(|nabla psi|/a*) nabla psi in real space (nonlinear step is pointwise)
- Take divergence via FFT
- Iterate to self-consistency

This is essentially what multigrid MOND solvers do, but the connection to perturbative mode-coupling kernels has not been formally established.

## 6. Effective Field Theory of Large-Scale Structure (EFTofLSS)

### 6.1 The Standard Framework

The EFTofLSS was introduced by Baumann, Nicolis & Senatore (2012) [arXiv:1206.2926] and developed extensively by Senatore, Zaldarriaga, Carrasco and others.

Key idea: On large scales, the effect of small-scale (UV) physics on the long-wavelength perturbations can be captured by a few effective parameters (speed of sound, viscosity). The framework:

1. Start with SPT at one-loop level
2. Add counterterms parameterized by c_s^2 (effective sound speed) to absorb UV sensitivity
3. The one-loop EFT power spectrum is:

```
P_EFT(k) = P_L(k) + P_22(k) + 2P_13(k) - 2 c_s^2 k^2 P_L(k)
```

The counterterm -2 c_s^2 k^2 P_L(k) absorbs the UV divergence that appears in P_13.

### 6.2 Comprehensive Review

**Ivanov (2022)**, "Effective Field Theory for Large Scale Structure" [arXiv:2212.08488] provides a thorough modern review covering the theoretical framework, practical implementation, and comparison with galaxy survey data.

### 6.3 Extension to Modified Gravity

**Bose et al. (2018)** and **Aviles & Cervantes-Cota (2017)** [arXiv:1606.02520] have extended EFTofLSS to modified gravity theories:

- The SPT kernels F_n, G_n are modified when the Poisson equation is changed
- For f(R) gravity and DGP braneworld models, modified kernels have been computed [arXiv:2210.00772]
- The EFT counterterms may need modification because the UV physics is different in modified gravity
- **One-loop power spectra have been computed for nDGP gravity and compared with N-body simulations**

### 6.4 Could EFTofLSS Be Adapted for MOND/DFD?

**This is conceptually possible but technically challenging:**

**Obstacles:**
1. EFTofLSS assumes a perturbative expansion around a homogeneous FRW background. In MOND, the background Hubble flow acceleration H*r may be comparable to a_0 at certain epochs, making the background itself MOND-affected
2. The SPT kernels assume a LINEAR Poisson equation. For MOND/DFD, the kernels must be re-derived from the nonlinear field equation
3. The UV counterterms depend on the nature of small-scale physics, which is fundamentally different when the gravitational operator is nonlinear
4. The perturbative expansion may not converge well because mode coupling is stronger (possibly O(delta) rather than O(delta^2))

**Opportunities:**
1. QUMOND could provide a useful intermediate framework: solve the linear Poisson equation perturbatively (standard EFTofLSS), then apply the nonlinear mu-correction
2. The EFT philosophy -- that short-distance physics can be absorbed into a few parameters -- may still hold, but the parameters will be mu-function-dependent
3. The Skordis-Zlosnik AeST theory has a well-defined cosmological perturbation theory at linear level; extending to one-loop would be a natural next step

**NO PUBLISHED PAPER has applied EFTofLSS specifically to AQUAL/MOND gravity.**

## 7. Summary: Literature Gaps Relevant to DFD

### 7.1 What Exists
- Full SPT framework with F_n, G_n kernels for Newtonian gravity (Bernardeau et al. 2002)
- N-body simulations of MOND cosmology (Nusser 2002; Llinares et al. 2008; nuHDM 2025)
- Linear cosmological perturbation theory for relativistic MOND (Skordis & Zlosnik 2021)
- EFTofLSS for some modified gravity theories (f(R), DGP) but not MOND
- Knowledge that MOND drives P(k) toward k^{-1} (Nusser 2002)
- Knowledge that MOND overproduces large-scale structure (nuHDM simulations)

### 7.2 What Does NOT Exist (Gaps DFD Could Fill)

1. **No perturbative mode-coupling kernels for a nonlinear Poisson equation** -- Nobody has derived the analog of F_2(k_1, k_2) for nabla.[mu(|nabla Phi|) nabla Phi] = source

2. **No analytic estimate of whether MOND mode coupling is O(delta) or O(delta^2)** -- The argument is clear qualitatively but nobody has shown it quantitatively

3. **No EFTofLSS for MOND/AQUAL** -- The framework has been extended to other modified gravity theories but not to theories with a nonlinear Poisson operator

4. **No computation of BAO evolution under a nonlinear Poisson equation** -- How does the MOND nonlinearity affect the BAO feature specifically?

5. **No formal connection between multigrid numerical solutions and perturbative kernels** -- The numerical and analytical approaches to the nonlinear Poisson equation have not been bridged

6. **No analytic nonlinear P(k) for any MOND-like theory** -- All nonlinear P(k) results are from N-body simulations, not perturbation theory

## 8. Recommendations for DFD P(k) Calculation

### 8.1 Suggested Approach: Perturbative Expansion of DFD Field Equation

Starting from:
```
nabla . [mu(|nabla psi| / a*) nabla psi] = -(8piG/c^2) rho
```

1. **Expand mu around background**: Let psi = psi_0 + psi_1 + psi_2 + ... where psi_0 is the homogeneous (or slowly varying) background field

2. **Derive DFD F_2 kernel**: The second-order perturbation psi_2 will involve a mode-coupling integral with a kernel F_2^DFD(k_1, k_2) that depends on:
   - The standard F_2 terms
   - Additional terms from mu' and mu'' evaluated on the background
   - The direction of the background field gradient (anisotropy)

3. **Compute P_22 and P_13 for DFD**: Integrate the DFD kernels against the linear P(k) to get the one-loop correction

4. **Add EFT counterterms**: Parameterize the UV sensitivity with DFD-specific counterterms

### 8.2 The QUMOND Shortcut

An alternative approach using QUMOND:
1. Solve the standard Newtonian problem including SPT mode coupling
2. Compute the "phantom dark matter" distribution from the Newtonian potential
3. Add the phantom dark matter contribution to get the total effective density
4. This avoids solving the nonlinear PDE perturbatively at the cost of some approximation

### 8.3 What to Compute First

The most impactful first calculation would be:
**Derive the DFD/MOND F_2 kernel and show explicitly that it has O(delta) contributions that are absent in standard gravity.** This would be a novel result with immediate implications for the P(k) problem.
