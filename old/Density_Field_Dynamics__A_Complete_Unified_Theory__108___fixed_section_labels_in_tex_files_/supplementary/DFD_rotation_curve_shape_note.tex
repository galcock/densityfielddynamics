\section*{Supplementary Note: Model-Independent SPARC Shape Test}
This package also includes a v3.2 integration of the model-independent SPARC interpolation-function shape analysis summarized in Section~\ref{sec:galactic-shape-paper}. The full standalone PRD-style manuscript was used to extract the numerical results reported there: $n_{\rm opt}=1.15\pm0.12$ (bootstrap 95\% CI $[1.00,1.50]$), DFD's $n=1$ inside the confidence interval, and Standard MOND's $n=2$ strongly disfavored in both free-$\Upsilon_\star$ and fixed-$\Upsilon_\star$ tests. This supplementary file is included so the master archive retains the dedicated rotation-curve shape material alongside the unified review source.
